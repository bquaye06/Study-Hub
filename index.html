Core Concepts in 10 Minutes

  • Java flow: write source code, compile to bytecode, run on JVM.
  • Variables: local, instance, static.
  • Primitives: byte, short, int, long, float, double, char, boolean.
  • Control flow: if/else, switch, while, do-while, for.
  • OOP fundamentals: class, object, constructor, encapsulation.
  • Collections overview: ArrayList for dynamic lists, Set for uniqueness, HashMap for key-value pairs.
  • Exception handling for fault-tolerant input and arithmetic operations.

Must-Know Syntax

// Input
Scanner sc = new Scanner(System.in);
int n = sc.nextInt();

// Condition
if (n >= 50) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}

// Loop
for (int i = 0; i < arr.length; i++) {
    sum += arr[i];
}

// Exception handling
try {
    int x = sc.nextInt();
} catch (InputMismatchException e) {
    System.out.println("Invalid input");
}

// Class + constructor
class Box {
    private double width;

    public Box(double width) {
        this.width = width;
    }
}

// File write
PrintWriter out = new PrintWriter("data.txt");
out.println("Hello");
out.close();

Common Traps

  • Comparing strings with == instead of .equals().
  • Forgetting degree-to-radian conversion in trig functions.
  • Using integer division when decimal output is needed.
  • Missing braces causing wrong control-flow grouping.
  • Not closing file writers/readers.

Exam Checklist

  1. Can you write input-process-output programs quickly?
  2. Can you choose the correct loop type by requirement?
  3. Can you defend against bad input with try/catch?
  4. Can you design a class with constructor + methods?
  5. Can you decide between array, ArrayList, Set, and HashMap?

Practical Coding Roadmap

A step-by-step path to build exam confidence through implementation.

Jump to: Week 1-2 Week 3-4 Week 5-6 Capstone

Week 1-2: Basics + Console Skills

  1. Create 10 mini programs on variables, arithmetic, and type conversion.
  2. Implement BMI calculator and grade classification with if/else.
  3. Practice Scanner input for int, double, and String in one flow.

Week 3-4: Strings + Loop Mastery

  1. Write string utility programs: count vowels, palindrome check, substring extraction.
  2. Implement 5 loop challenges: sum range, multiplication table, menu loop, sentinel input, pattern printing.
  3. Add basic input validation and user re-entry logic.

Week 5-6: Exceptions + Data Structures + OOP

  1. Refactor prior programs with try/catch for robust behavior.
  2. Build one project each using array, ArrayList, Set, and HashMap.
  3. Create classes (Student, Course, Result) with constructors and methods.
  4. Apply encapsulation, static members, and method decomposition.

Capstone: Student Results Manager

Build one integrated application combining all major course skills.

  • Input student records from console or file.
  • Validate data with exception handling.
  • Store records in ArrayList and HashMap.
  • Use OOP classes for clean architecture.
  • Optional: add a Swing GUI and export runnable JAR.
